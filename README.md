# Mini RAG System (Vector Search)

This project implements a small Retrieval-Augmented Generation (RAG) system
using embeddings and vector search.

## Project Goals
- Build a basic vector search pipeline
- Store and retrieve embeddings
- Ground LLM responses in retrieved context
- Demonstrate core RAG concepts for applied AI roles

## Versions
- **v1**: Initial RAG implementation (embeddings + vector search + grounded answers)
